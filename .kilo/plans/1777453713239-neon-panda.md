# Plan: Perbaikan Aturan Assign Instruktur Berdasarkan Keahlian

## Ringkasan

Menambahkan validasi keahlian saat meng-assign instruktur ke sesi/blok materi, serta
menambahkan fitur hapus assignment per-sesi dan batch delete.

---

## 1. Validasi Keahlian pada Server Actions

### 1.1 `assignInstructorToBlock` — `src/server/actions/jadwal-otomatis/assignments.ts`

- Sebelum loop insert, cari `programId` dari `kelasPelatihan`
- Query `instructorExpertise` untuk cek apakah instruktur punya keahlian
  `(instructorId, programId, materiBlock)`
- Jika tidak ditemukan (atau ada tapi `level` tidak dikenal), return error:
  `{ ok: false, error: "Instruktur tidak memiliki keahlian untuk blok materi "{materiBlock}". Tambahkan keahlian terlebih dahulu." }`
- Jika ditemukan, lanjutkan loop seperti biasa

### 1.2 `assignInstructorToSession` — file yang sama

- Sebelum insert, ambil `classSessions` → `materiName` dan join ke `kelasPelatihan` → `programId`
- Query `instructorExpertise` dengan `(instructorId, programId, session.materiName)`
- Jika tidak cocok, return error serupa

---

## 2. Fungsi Hapus Assignment

### 2.1 `unassignInstructorFromSession(assignmentId: string)` — file baru/assignments.ts

- `requirePermission("jadwalUjian", "manage")`
- `DELETE FROM sessionAssignments WHERE id = assignmentId`
- `revalidatePath("/jadwal-otomatis")`
- Return `{ ok: true }`

### 2.2 `unassignInstructorFromBlock(kelasId: string, materiBlock: string)` — file yang sama

- `requirePermission("jadwalUjian", "manage")`
- Cari semua `classSessions` di kelas tersebut dengan `materiName = materiBlock`
- `DELETE FROM sessionAssignments WHERE sessionId IN (...sessions.map(s => s.id))`
- `revalidatePath("/jadwal-otomatis")`
- Return `{ ok: true, deletedCount }`

### 2.3 `bulkUnassignInstructors(assignmentIds: string[])` — file yang sama

- `requirePermission("jadwalUjian", "manage")`
- Zod: array of strings min 1, max 200
- `DELETE FROM sessionAssignments WHERE id IN (assignmentIds)`
- `revalidatePath("/jadwal-otomatis")`
- Return `{ ok: true, deletedCount }`

## 3. Perubahan UI — `JadwalDetail.tsx`

### 3.1 Filter dropdown instruktur berdasarkan keahlian

**Current behavior:** Dropdown menampilkan semua instruktur aktif, diurutkan
berdasarkan rekomendasi (yang tanpa keahlian tetap muncul di urutan bawah).

**New behavior:**
- Ketika `assignBlock` dipilih **dan** `recommendations` sudah ter-load:
  - Dropdown instruktur HANYA menampilkan instruktur yang ada di array `recommendations`
  - Jika `recommendations.length === 0`, dropdown disable + tampilkan teks
    "Tidak ada instruktur dengan keahlian untuk blok ini"
- Ketika `assignBlock` kosong: dropdown tetap disable (karena tombol Assign
  juga disabled kalau `!assignBlock`)
- Ketika `assignBlock` dipilih tapi recommendations masih loading:
  - Dropdown disable, tampilkan "Memuat rekomendasi..."

**Implementasi:** filter `orderedInstructors` intersection dengan `recommendations`:

```tsx
const availableInstructors = useMemo(() => {
  if (!assignBlock || loadingRecommendations || recommendations.length === 0) return [];
  const recommendedIds = new Set(recommendations.map(r => r.instructorId));
  return orderedInstructors.filter(i => recommendedIds.has(i.id));
}, [orderedInstructors, recommendations, assignBlock, loadingRecommendations]);
```

Ubah map di SelectContent dari `orderedInstructors` jadi `availableInstructors`.
Sesuaikan placeholder SelectTrigger berdasarkan state.

### 3.2 Tombol hapus per-sesi

Tambahkan kolom aksi di tabel jadwal:

- Tampilkan tombol/ikon hapus (`Trash2`) pada baris sesi yang memiliki assignment (`assignBySession.get(session.id) !== undefined`)
- Konfirmasi dengan `confirm()` atau dialog sebelum menghapus
- Panggil `unassignInstructorFromSession(assignmentId)` + `router.refresh()`

### 3.3 Batch delete dengan checkbox

- Tambah kolom checkbox di setiap baris sesi yang punya assignment
- Checkbox di header untuk select all (hanya yang punya assignment)
- Tombol "Hapus Terpilih" muncul jika ada item tercentang
- Konfirmasi batch delete
- Panggil `bulkUnassignInstructors(selectedIds)`
